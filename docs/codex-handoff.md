- 直前の変更で `LINEで通知` を月ヘッダー右端へ移動した結果、`justify-between` ベースの並びになっていた

- ヘッダー全体を左右分配で配置していたため、中央の月ナビが「レイアウト上の中央」ではなく「残り幅の中央」になっていた

### 仮説
- 将来左側にも補助アクションが増えるなら、今回の 3 カラム構成のほうが拡張しやすい
