# Codex Handoff Prompt

作業の最後に、必ず ChatGPT 引き継ぎ用サマリーを日本語で作成してください。

## ルール
- 推測ではなく、実際に確認した内容ベースで書く
- 長すぎず、次の判断に必要な情報は残す
- 変更ファイルは必ずパス付きで書く
- 実行したコマンドだけを書く
- 未確認事項と仮説は分ける
- Markdown形式で出力する
- 最後に「ChatGPTにそのまま貼れる依頼文」を1つ入れる

## 出力形式

# Codex作業サマリー

## 1. 今回の目的
- 何をしようとしたか

## 2. 確認した状況
- 実際に確認したコードや挙動
- 関連ファイル
- 現状の問題点

## 3. 原因
### 確定
- 断定できる原因

### 仮説
- まだ仮説段階のもの

## 4. 実施した変更
- 変更したファイル一覧
- 各ファイルで何を変えたか
- 破壊的変更があるなら明記

## 5. テスト・確認結果
- 実行したコマンド
- 成功したこと
- 失敗したこと
- 未実施の確認

## 6. 未解決事項
- まだ残っている問題
- 判断待ちの論点
- UI/文言/設計で迷っている点

## 7. 次にやるなら
- 優先度順に 3つまで

## 8. ChatGPTに相談したいこと
- ChatGPTに整理・判断・提案してほしい論点

## 9. ChatGPTに次に頼む依頼文
- そのまま貼れる依頼文を1つ
